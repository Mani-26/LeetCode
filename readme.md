

# LeetCode Solutions

A collection of solutions for LeetCode problems. Each file corresponds to a LeetCode problem, named after the problem's title for easy reference.

## Table of Contents
- [About](#about)
- [How to Use](#how-to-use)
- [Contributing](#contributing)
- [License](#license)

## About
This repository contains solutions to various LeetCode problems, implemented in Java. Solutions are aimed at helping others learn different problem-solving techniques.

## How to Use
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Mani-26/LeetCode.git
   ```

2. **Find the solution file**:
   Each problem solution is in a separate file, usually named after the problem, e.g., `Two_Sum.java`.

3. **Run the code**:
   Open the file to view the code and run it in your IDE.

## Contributing
Contributions are welcome! Please follow these steps:

1. **Fork the repository**.
2. **Create a new branch** for your solution:
   ```bash
   git checkout -b feature/new-solution
   ```
3. **Add your solution file** with a meaningful name and include comments to explain your code.
4. **Commit your changes** and push the branch:
   ```bash
   git push origin feature/new-solution
   ```
5. **Open a Pull Request** with a brief description of your solution.
